#README For GitHub

Python clone of Minecraft

Controls:

	W - Forwards
	S - Backwards
	A - Strafe Left
	D - Strafe Right
	G - Stop Flying
	H - Start Flying
	Z - Fly Down
	SPACE - Jump/Fly Up
	ENTER - Save Player Location
	MOUSE - Look Around

Start Game:
	Use the RUN.bat command, or start main.py with python.
